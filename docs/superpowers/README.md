# Design docs and implementation plans

Working documents produced while building the platform — design specs (`specs/`) and step-by-step implementation plans (`plans/`), written with and for AI agents as part of the dev workflow this platform exists to automate.

They are kept as a **historical record** of how and why things were built, dogfooding included. Expect them to reference the sibling `agentops-engine` repo and to describe states the manifests have since moved past — when a spec and the manifests disagree, the manifests win.
